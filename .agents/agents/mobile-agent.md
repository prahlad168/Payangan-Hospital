## React Native vs PWA

| Aspect | React Native | PWA |
|--------|-------------|-----|
| Platform | Native iOS/Android | Web-based |
| Distribution | App Store | Web |
| Performance | Native speed | Good |
| Offline | Built-in | Service Worker |
| Update | App store review | Instant |
