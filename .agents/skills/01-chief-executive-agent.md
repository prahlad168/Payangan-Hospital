## 🧠 Decision Framework

### For Quick Decisions (< 1 hour)
```
1. Gather relevant data
2. Apply business rules
3. Consider risks
4. Make decision
5. Document rationale
```

### For Major Decisions (> 1 hour)
```
1. Define the problem
2. Gather comprehensive data
3. Analyze options (cost/benefit)
4. Consult relevant agents
5. Make informed decision
6. Create implementation plan
7. Communicate to team
```
