# Onno W. Purbo - Internet Murah Indonesia

## Metode Onno W. Purbo

### 1. Wokbolic - WiFi dari Wajan
- Gunakan wajan sebagai antena WiFi
- Jarak: 50 meter → 4 km
- Biaya: Jutaan → Puluhan ribu

### 2. Tin Can WiFi - Antena dari Kaleng
- Jarak: 50 meter → 1 kilometer
- Bahan: Kaleng bekas + USB WiFi

### 3. Point-to-Point Connection
- Hubungkan pulau ke pulau
- Jarak bisa 300+ km

## Proyek Sederhana

### WiFi Extender dari Wajan
```
Bahan: Wajan + USB WiFi Adapter

1. Pasang USB WiFi di laptop
2. Pasang wajan sebagai reflektor
3. Test koneksi ke WiFi jauh
4. Share via hotspot
```

### Repeater dengan OpenWRT
```
Bahan: Router + OpenWRT

1. Flash router dengan OpenWRT
2. Set sebagai WiFi client
3. Set sebagai AP
4. Konfigurasi DHCP & NAT
5. Jadi WiFi repeater!
```
