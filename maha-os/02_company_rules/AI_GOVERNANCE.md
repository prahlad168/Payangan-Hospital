## 3.2 Communication Channels

| Channel | Purpose | Response Time |
|---------|---------|---------------|
| **Direct Message** | Urgent tasks | < 15 minutes |
| **Task Queue** | Regular tasks | < 1 hour |
| **Report Channel** | Status updates | < 4 hours |
| **Escalation Channel** | Issues/Blockers | < 1 hour |
| **Knowledge Channel** | Knowledge sharing | Daily batch |

## 3.3 Status Updates

| Status | Meaning | Action Required |
|--------|---------|----------------|
| RECEIVED | Task received | Acknowledge |

| BLOCKED | Cannot proceed | Escalate immediately |
| QA_REVIEW | Awaiting QA | Send to QA agent |
| COMPLETED | Done | Report results |

```
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
NOT JUST "CAN WE?" BUT ALSO:
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
```

1. **Is this LEGAL?**
   - Does it comply with Indonesian law?
   - Does it follow company policies?
   - Are there regulatory requirements?

2. **Is this FINANCIAL?**
   - What's the cost?
   - What's the expected return?
   - Is it within budget?

3. **Is this TECHNICAL?**
   - Can we execute it?
   - Do we have resources?
   - What's the complexity?

4. **Is this OPERATIONAL?**
   - Does it fit our processes?
   - What's the impact on other teams?
   - Can we scale it?

5. **Is this SECURE?**
   - Any data security risks?
   - Any cybersecurity concerns?
   - Access control issues?

6. **Is this CUSTOMER-FRIENDLY?**
   - What's the impact on customers?
   - Does it improve or hurt experience?
   - Any compliance risks?

7. **What's the BUSINESS VALUE?**
   - Strategic alignment?
   - Competitive advantage?
   - Long-term benefits?

8. **What's the RISK?**
   - What's the worst case?
   - Can we mitigate it?
   - Is it acceptable?

9. **What's the ROI?**
   - Payback period?
   - Expected return?
   - Risk-adjusted return?

10. **What's our CONFIDENCE?**
    - How sure are we?
    - What's the data?
    - What could go wrong?

| KPI | Description | Target | Frequency |
|-----|-------------|--------|-----------|
| **Tasks Completed** | Number of tasks finished | Department target | Daily |
| **Tasks Failed** | Number of failed tasks | < 5% | Daily |
| **Quality Score** | Output quality (1-10) | > 8 | Weekly |
| **Speed Score** | Execution speed (1-10) | > 7 | Weekly |
| **Knowledge Added** | New KB entries created | > 3/week | Weekly |
| **SOP Improved** | SOP updates made | > 1/week | Weekly |
| **Automation Created** | Automations implemented | > 1/month | Monthly |
| **Revenue Impact** | Revenue influenced | Department target | Monthly |
| **Cost Reduction** | Costs saved | > Rp 500K/month | Monthly |
| **Innovation Score** | New ideas generated | > 5/week | Weekly |
| **Customer Satisfaction** | Client satisfaction | > 90% | Monthly |

## 5.2 KPI Calculation

```
Quality Score = (Accuracy × 0.3) + (Completeness × 0.3) + 
                (Timeliness × 0.2) + (Compliance × 0.2)

Speed Score = (Expected Time / Actual Time) × 10
             (cap at 10, floor at 1)

Overall Performance = (KPI Average × 0.6) + (Quality × 0.2) + 
                      (Learning × 0.2)
```

## 5.3 Performance Ratings

| Rating | Score Range | Action |
|--------|-------------|--------|
| Exceptional | 9-10 | Promote consideration |
| Exceeds | 8-9 | Bonus eligibility |
| Meets | 7-8 | Standard |
| Developing | 5-7 | Improvement plan |
| Below | < 5 | Performance review |
