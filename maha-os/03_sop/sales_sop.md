### 4.2 Lead Qualification (STAGE 2)

#### Qualification Framework (BANT):
```
в””в”Җв”Җ B - Budget
в””в”Җв”Җ  в”Ӯ в”Ӯ  Is budget aligned?
в””в”Җв”Җ  в”Ӯ в”Ӯ  Budget range: <500K / 500K-5M / >5M
в””в”Җв”Җв”Ӯ
в””в”Җв”Җ A - Authority
в””в”Җв”Җ  в”Ӯ в”Ӯ  Is decision maker?
в””в”Җв”Җ  в”Ӯ в”Ӯ  Level: IC / Manager / Director / VP / C-Level
в””в”Җв”Җв”Ӯ
в””в”Җв”Җ N - Need
в””в”Җв”Җ  в”Ӯ в”Ӯ  Is there a clear pain point?
в””в”Җв”Җ  в”Ӯ в”Ӯ  Need alignment: 1-10 score
в””в”Җв”Җв”Ӯ
в””в”Җв”Җ T - Timeline
в””в”Җв”Җ  в”Ӯ в”Ӯ  When do they want to buy?
в””в”Җв”Җ  в”Ӯ в”Ӯ  Timeline: Immediate / 30 / 60 / 90+ days
```

#### Qualification Score:
```
в””в”Җв”Җ SCORE 80-100: HOT LEAD
в””в”Җв”Җ  в”Ӯ в”Ӯ  Immediate follow-up (within 1 hour)
в””в”Җв”Җ  в”Ӯ в”Ӯ  Schedule demo within 24 hours
в””в”Җв”Җв”Ӯ
в””в”Җв”Җ SCORE 50-79: WARM LEAD
в””в”Җв”Җ  в”Ӯ в”Ӯ  Follow-up within 24 hours
в””в”Җв”Җ  в”Ӯ в”Ӯ  Nurture sequence
в””в”Җв”Җв”Ӯ
в””в”Җв”Җ SCORE < 50: COLD LEAD
в””в”Җв”Җ  в”Ӯ в”Ӯ  Add to nurture sequence
в””в”Җв”Җ  в”Ӯ в”Ӯ  Review every 30 days
```

### 4.3 Outreach & Follow-up

#### Response Time SLA:
```
в””в”Җв”Җ Hot Lead: < 1 hour
в””в”Җв”Җ Warm Lead: < 4 hours
в””в”Җв”Җ Cold Lead: < 24 hours
в””в”Җв”Җ Form Submit: < 1 hour
```

#### Follow-up Sequence:
```
в””в”Җв”Җ Day 1: Initial outreach (email + LinkedIn)
в””в”Җв”Җ Day 3: Follow-up email #1
в””в”Җв”Җ Day 7: Follow-up email #2
в””в”Җв”Җ Day 14: Break-up email
в””в”Җв”Җ Day 30: Re-engagement
в””в”Җв”Җ Day 60: Final attempt
```

### 4.4 Proposal & Quotation (STAGE 3)

#### Proposal Structure:
```
в””в”Җв”Җв”Ӯ 1. Executive Summary
в””в”Җв”Җв”Ӯ 2. Problem Statement
в””в”Җв”Җв”Ӯ 3. Proposed Solution

в””в”Җв”Җв”Ӯ 6. Terms & Conditions

```

#### Pricing Tiers:
```

в””в”Җв”Җ PROFESSIONAL: Rp 1.000.000 - 5.000.000
в””в”Җв”Җ ENTERPRISE: Rp 5.000.000+
```

### 4.5 Negotiation (STAGE 4)

#### Negotiation Guidelines:
```
в””в”Җв”Җ Never discount more than 20% without approval
в””в”Җв”Җ Always offer alternative (smaller package, longer timeline)
в””в”Җв”Җ Focus on value, not price
в””в”Җв”Җ Know your minimum acceptable price
в””в”Җв”Җ Get something in return for any discount
```

### 4.6 Closing (STAGE 5)

#### Closing Techniques:
```
в””в”Җв”Җ Assumptive Close: "We'll start Monday then?"
в””в”Җв”Җ Urgency Close: "This price expires Friday"
в””в”Җв”Җ Summary Close: Recap all value, ask for decision
в””в”Җв”Җ Direct Close: "Are you ready to sign?"
```

### 4.7 Post-Sale Handover

#### Handover Checklist:
```
в””в”Җв”Җ [ ] Send welcome email
в””в”Җв”Җ [ ] Add to CRM with "Won" status
в””в”Җв”Җ [ ] Create project in PM tool

в””в”Җв”Җ [ ] Set milestone reminders

```

## LEARNING POINTS

### Best Practices:
- Always follow up within SLA
- Use CRM to track everything
- Personalize every outreach
- Focus on value, not price
- Ask for referrals at closing

### Common Mistakes:
- Not following up consistently
- Poor lead qualification
- Generic outreach
- Price-focused selling
- No post-sale follow-up
